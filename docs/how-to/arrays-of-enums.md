# Use arrays of enums

You can combine arrays with enums, this way:

```ts
import { Entity, Column } from "@thothom/core";

import { RoleEnum } from "./role.enum";

@Entity()
export class ExampleEntity {
  // ...

  @Column({
    type: String,
    enum: RoleEnum,
  })
  public roles: Array<RoleEnum>;

  // ...
}
```
